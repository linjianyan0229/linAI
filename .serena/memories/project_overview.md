# linAI 项目概览
- 项目类型：VitePress 文档站点
- 主要目的：当前仓库用于承载文档站内容，但现阶段仍基本保持 VitePress 默认初始化模板。
- 当前状态：站点标题为 `linAIdocs`，首页、导航、侧边栏、示例页面均为默认示例内容，尚未体现明确业务领域。
- 技术栈：Markdown、VitePress、TypeScript 配置文件（`config.mts`）。
- 入口位置：文档入口为 `docs/index.md`，站点配置位于 `docs/.vitepress/config.mts`，npm 脚本位于 `package.json`。
- 代码结构非常精简，目前几乎没有自定义组件、主题扩展、业务脚本或测试代码。
