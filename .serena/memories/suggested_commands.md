# 推荐命令
## 项目命令
- `npm run docs:dev`：本地启动 VitePress 文档站开发服务器
- `npm run docs:build`：构建文档站静态产物
- `npm run docs:preview`：预览构建后的站点

## Windows 常用辅助命令
- `git status`：查看工作区状态
- `git diff`：查看改动
- `Get-ChildItem`：列目录（PowerShell 中相当于 `ls`）
- `Set-Location <path>`：切换目录（PowerShell 中相当于 `cd`）
- `rg <pattern>`：全文检索（若本机已安装 ripgrep）

## 说明
- 仓库中当前未发现 lint、format、test 相关脚本。
- 如需运行项目，先确保已安装 VitePress 相关依赖。
